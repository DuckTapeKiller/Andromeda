# Andromeda Theme

Andromeda is a calm, stellar-inspired theme for Obsidian. It combines a 'Scattered Dust' background pattern with tactile UI interactions and a focus on readability using serif typography for the main text.

![Andromeda Theme Screenshot](screenshot.png)

## Features

* **Atmospheric Modes:**
    * **Light Mode:** 'Old Paper & Deep Ink' palette using oatmeal backgrounds with deep green accents.
    * **Dark Mode:** 'Deep Space' palette using dark grey surfaces with teal accents.
* **Scattered Dust Background:** A subtle radial gradient pattern applied to the workspace background to mimic star clusters or dust.
* **Tactile Interactivity:** Buttons and clickable icons utilise a 'pop' transition, lifting and scaling slightly on hover.
* **Typography:**
    * **Editor:** Uses serif fonts (Lora, Merriweather, or Georgia) for a classic reading experience.
    * **UI:** Uses system sans-serif fonts for clear navigation.
* **EnsÅ Dot Folders:** Replaces standard folder arrows with a minimalist dot system that changes colour and scale to indicate open/closed states.
* **Header Palettes:** Distinct, muted earth tones for headers in light mode and pastel planetary tones for dark mode.

## Installation

1.  Download the `theme.css` file.
2.  Open your Obsidian vault folder.
3.  Navigate to `.obsidian/themes/`.
4.  Create a folder named `Andromeda` and place the `theme.css` file inside.
5.  Open Obsidian Settings > Appearance and select **Andromeda**.
